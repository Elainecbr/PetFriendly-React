# Roteiro de Video - Entrega MVP (maximo 5 minutos)

## Objetivo

Apresentar de forma direta os pontos exigidos na avaliacao: objetivo da aplicacao, componentizacao, navegacao com hooks, usabilidade e responsividade.

## Tempo sugerido por bloco

### Bloco 1 (30s) - Objetivo do projeto

- Problema que a aplicacao resolve.
- Contexto PetFriendly.
- Resultado esperado para o usuario.

### Bloco 2 (90s) - Componentizacao (4+ componentes)

Mostrar os componentes reutilizados e onde aparecem:

- Header
- SearchBar
- ServiceCard
- FeedbackAlert
- LoadingSpinner

Explicar rapidamente a vantagem de reaproveitamento.

### Bloco 3 (60s) - Navegacao e roteamento

Demonstrar:

- Rotas principais.
- Rota dinamica de detalhes.
- Rota 404.
- Uso de useNavigate, useParams e useLocation.

### Bloco 4 (60s) - Usabilidade

Mostrar em funcionamento:

- Loading.
- Mensagem de nenhum item encontrado.
- Mensagem de sucesso/erro no agendamento.
- Busca rapida na Home para Servicos.

### Bloco 5 (45s) - Responsividade

- Troca de viewport desktop para mobile.
- Comentar que layout permanece legivel e funcional.

### Bloco 6 (15s) - Encerramento

- Onde esta o repositorio.
- Onde esta o deploy (quando publicado).

## Checklist antes de gravar

- Rodar npm run dev.
- Deixar as paginas abertas para transicao rapida.
- Testar fluxo de busca e agendamento.
- Confirmar que a rota 404 esta funcionando.
- Falar de forma objetiva, sem ler integralmente.
